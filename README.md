# Scheduled-Multi-Agent-Auto-Run-Agent-Workflow-Engine
Allow users to schedule agent runs (e.g., every morning) for updates or monitoring.


This project is a scheduled multi-agent workflow engine that automates recurring AI tasks. It is built using three core agents: a Research Agent that collects and generates topic-specific information, a Summary Agent that condenses the research into a concise format, and an Insight Agent that produces key takeaways, trends, or action-oriented observations. These agents run sequentially as part of a predefined workflow, triggered automatically by a scheduler at selected times. Each workflow run is stored as a session in the database, and the final outputs are displayed in a Streamlit dashboard so users can review previous automated runs and monitor recurring analysis over time.
